## Editor

| Action            | Shortcut |
| ----------------- | -------- |
| Run project       | ⌘ R      |
| Run current scene | ⌘ ⇧ R    |
| Stop running      | Escape   |
| Save scene        | ⌘ S      |
| Save all          | ⌘ ⌥ S    |
| Open scene        | ⌘ O      |
| New scene         | ⌘ N      |
| Undo              | ⌘ Z      |
| Redo              | ⌘ ⇧ Z    |
| Find in scene     | ⌘ F      |
